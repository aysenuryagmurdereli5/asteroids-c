# Asteroids Game (C & SDL2)

This is a simple 2D Asteroids game clone written in C language. 
It uses the SDL2 library for graphics and window management.
This project is developed for the YZM104 Programming II course at Kocaeli University.

## Features (Planned)
* Spaceship movement with vector math (velocity and acceleration)
* Shooting mechanics
* Asteroid splitting and collision detection
* Score and health system

## Requirements
To compile and run this game on Windows, you need:
* MSYS2 / MinGW-w64
* GCC Compiler
* Make
* SDL2 Library

1. Open your terminal (MSYS2 or VS Code terminal).
